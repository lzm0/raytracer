# raytracer
A simple ray tracer written in rust.

## Features
 - [x] Diffuse
 - [x] Multithreading
 - [ ] Metal
 - [ ] Glass
 - [ ] DOF

## Usage

```
cargo run --quiet --release > image.ppm
```
